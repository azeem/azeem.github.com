Webvs

Code on GitHub | Demo

A javascript/WebGL clone of the popular Winamp Advanced Visual Studio. Desinged as a library capable of reading JSON representations of AVS presets and rendering them on a WebGL canvas. It has a built-in AVS expression to Javascript/GLSL convertor, thus allowning presets code from AVS to run directly.

Knowledge Based Text Retrieval System

Code on Bitbucket

A Text Retrieval System that models documents and queries as graphs of connected concepts in a Knowledge Base. A proof of concept system, that uses Wikipedia article titles and article categorizations for concept matching and concept graph generation. Concepts in document graphs were ranked using a Page-Rank like algorithm.
